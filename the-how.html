How to Use Kelvingrover

Overview

Kelvingrover is designed to give you a number of routes of varying difficulty. Some routes have been made with consideration for those with restricted mobility - these can be found under the "Routes for mobility-assisted" entry in the "Walking Routes" tab at the top.

We've also included a number of cycle routes, which mainly stick to the wider paths of the park. A map of landmarks can be found at the bottom of all pages.

The Pages

Each of the route pages consists of 4 main elements: the page description; a map with buttons to select routes; an elevation map to give a general overview of the steepness of the routes; a map of landmarks located around Kelvingrove Park.

If you have already read the page description, you can dismiss it by clicking on the title of the page. The page description should then disappear, allowing for a more streamlined page.

To select a route, simply click on one of the numbered buttons above the routes map. Each of these buttons is assigned to a unique route around the park, which will be displayed on the map when clicked. An elevation chart will also pop up underneath the routes map to give a general overview of the route's intensity. This chart is not designed to give you an exact elevation at a given point, as we thought that giving an oversight of the track's overall variance was more important in deciding whether you should attempt it.

Finally, at the bottom of the page is a separate map of Kelvingrove Park with a number of markers placed on it. These markers represent landmarks throughout the park. You can click on any of these markers to open a popup with the landmark's name and a short description.
